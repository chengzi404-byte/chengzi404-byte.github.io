安装指南

简单几步，快速安装Phoenix Editor

Windows 安装

  1. 1

    从下载页面获取 PhoenixEditor-Setup.exe 安装文件

  2. 2

    双击运行安装文件，跟随安装向导步骤进行安装

  3. 3

    安装完成后，可在开始菜单或桌面上找到Phoenix Editor的快捷方式

macOS 安装

  1. 1

    从下载页面获取 PhoenixEditor.dmg 镜像文件

  2. 2

    双击DMG文件，将Phoenix Editor拖拽到应用程序文件夹中

  3. 3

    从应用程序文件夹启动Phoenix Editor，首次运行可能需要在系统偏好设置中允许运行

Linux 安装

  1. 1

    对于Ubuntu/Debian系统，获取 phoenix-editor.deb 安装包

  2. 2

    打开终端，运行命令: sudo dpkg -i phoenix-editor.deb

  3. 3

    安装完成后，可以从应用程序菜单启动，或在终端中运行 phoenix-editor 命令

使用方法

快速上手Phoenix Editor的基本操作

创建和打开文件

  • 点击菜单栏的"文件" -> "新建"创建新文件
  • 使用快捷键Ctrl+N (Windows/Linux) 或 Cmd+N (macOS)创建新文件
  • 点击"文件" -> "打开"选择要编辑的文件
  • 使用快捷键Ctrl+O 或 Cmd+O打开文件

保存文件

  • 点击菜单栏的"文件" -> "保存"保存当前文件
  • 使用快捷键Ctrl+S 或 Cmd+S保存文件
  • 使用"文件" -> "另存为"或快捷键Ctrl+Shift+S 另存文件
  • Phoenix Editor会自动保存您的编辑内容，防止意外丢失

运行代码

  • 点击菜单栏的"运行" -> "运行当前文件"
  • 使用快捷键Ctrl+R 或 Cmd+R运行代码
  • 运行结果会显示在底部的输出面板中
  • 可以使用Ctrl+C 终止正在运行的程序

使用AI辅助

  • 点击右侧面板的"AI助手"打开AI功能
  • 选中代码后右键选择"解释代码"获取代码解释
  • 在AI输入框中提问，获取编程相关帮助
  • 使用"优化代码"功能获取代码改进建议
